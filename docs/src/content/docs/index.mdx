## Three lines

```ts
import lingua from "@axiomantic/llmlingua-2";

const { compressed, reverseMap } = await lingua.compress(longText, { targetRatio: 0.5 });
const restored = await lingua.decompress(compressed, reverseMap);
```

`compress` runs the model and drops low-importance tokens. `decompress` reads the
faithful original out of `reverseMap` — see
[Decompression semantics](/llmlingua-2/guides/decompression-semantics/) for the contract.
