A typed, ESM-first wrapper around the [LLMLingua-2](https://github.com/microsoft/LLMLingua)
ONNX export. Runs entirely in Node (≥20) via `@huggingface/transformers`; no Python,
no native bindings. Bring-your-own peer dependency on `@huggingface/transformers@^3`.

## Three lines

```ts
import lingua from "@axiomantic/llmlingua-2-js";

const { compressed, reverseMap } = await lingua.compress(longText, { targetRatio: 0.5 });
const restored = await lingua.decompress(compressed, reverseMap);
```
